NAME

Date.getDow

FUNCTION

The Javascript method Date.getDow returns the day of the week as a string.

If logical true is passed for the first parameter, a 3-digit abbreviation is returned, using 'Thu' for Thursday.

INPUTS

boolean - If true, returns a 3-digit abbreviation

SOURCE

*
Date.prototype.getDow = function(makeItShort) {
    var days = ['Sunday','Monday','Tuesday','Wednesday'
        ,'Thursday','Friday','Saturday'
    ];
    var retval = days[this.getDay()];
    if(makeItShort) {
        return retval.slice(0,3);
    }
    return retval;
}
