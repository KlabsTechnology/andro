NAME

x6Data

FUNCTION

The PHP function x6Data takes a variable and sends it to the browser. After the page is loaded, or the JSON call is processed, the data will appear as a property of the x6.data object.

INPUTS


  • string - name of data element
  • mixed - the value or array

SOURCE

function x6Data($name,$data) { 
    $script = "\nx6.data.$name = ".json_encode_safe($data).";";
    x6Script($script);
}
