NAME

bufferedRender

FUNCTION

The method bufferedRender rendered this androHtml object in a buffer, instead of directly outputing it to the browser.

SOURCE

function bufferedRender($parentId='',$singleQuotes=false) {
    ob_start();
    $this->render($parentId,$singleQuotes);
    return ob_get_clean();
}
