NAME

hidden

FUNCTION

The method hidden adds a hidden value to this androHtml object. A hidden variable is stored in an input html element that has the html property 'type' set to 'hidden'. This enables the passing of variables back and forth from the server to the client browser after refreshes.

INPUTS

string $name - Name of the hidden variable string $value - Value for the hidden variable

SOURCE

function hidden($name,$value = '') {
    $h = $this->h('input');
    $h->hp['type'] = 'hidden';
    $h->hp['id']   = $name;
    $h->hp['name'] = $name;
    $h->hp['value']= $value;
    return $h;
}
