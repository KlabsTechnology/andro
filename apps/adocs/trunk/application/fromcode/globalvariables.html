NAME

Global_Variables

FUNCTION

Andromeda provides some wrapper that allow you to set and retrieve PHP global variables without risking a collission with framework global variables.

Use PHP API/vgaSet to set a variable, and PHP API/vgaGet to retrieve a variable.

The framework uses the corresponding functions PHP API/vgfSet and PHP API/vgfGet.
