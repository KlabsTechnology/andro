NAME

androHtml.hiddenInputs

FUNCTION

The PHP method hiddenInputs adds an invisible div to the current node and fills it with inputs for provided table. These can cloned (using jQuery) in browser-side code to place inputs on-the-fly anywhere on the screen.

The function returns a reference to the div. The div contains an associative array indexed on column name that contains references to the inputs.

INPUTS

mixed - either a table name or a data dictionary array reference.

RETURNS

reference - reference to the invisible div.
