NAME

render

FUNCTION

The method render renders this androHtml object. It builds all of the html code based on the objects attributes, children elements, parent elements, etc. Render directly outputs all html out to the browser. User bufferedRender to get the html as a string instead of outputting to the browser.

INPUTS

string $parentId - parent id for this androHtml object
