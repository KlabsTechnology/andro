NAME

androHtmlTabs.addTab

FUNCTION

This PHP class method addTab is the basic method of the androHTMLTabs class, call this function once for each tab you wish to add to your tabbar.

INPUTS

- $caption string, becomes both caption and ID

RETURNS

- androHTML, reference to a div where you can put content for the new tab.
