NAME

aFromgp

FUNCTION

The PHP function aFromGP captures zero or more GET/POST variables and returns an associative array. The function is called by provided a prefix string. Every GET/POST variable whose name begins with that prefix is captured and returned. The keys in the associative array do not contain the prefix, the prefix is stripped.

INPUTS

string prefix

RETURNS

associative array
