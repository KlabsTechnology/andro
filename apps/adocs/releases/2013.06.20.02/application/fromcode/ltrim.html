NAME

String.ltrim

FUNCTION

The Javascript function ltrim removes leading spaces from a string (spaces on the left side).

EXAMPLE

Example usage: 

     var x = '  abc  ';
     var y = x.ltrim();  // returns 'abc  '

SOURCE

String.prototype.ltrim = function() {
    return this.replace(/^\s+/,"");
}
