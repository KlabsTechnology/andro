NAME

tabFocus

FUNCTION

The PHP method tabFocus does exactly the same thing as tabIndex, with one additional action. When the page loads, this object will start out with focus.

Calling this method more than once while building a page causes focus to begin on the last object that made the call.

The first time you call tabIndex, it acts like a call to tabFocus, so there is no reason to ever call tabFocus unless you want focus to begin somewhere other than the first tabbable object.

INPUTS

int (optional) starting value. If this value is supplied, the object will get this value for its tabIndex, and subsequent calls will increment from there.

EXAMPLE

Here is an example: 

     <?php
     $div = html('div');
     $input = $div->h('input');
     $input->tabFocus();   // first input should get focus
     // more poperty settings...
     $input = $div->h('input'); // reuse var, make another input
     $input->tabIndex();
     ?>
