NAME

x6HTML

FUNCTION

The PHP function x6HTML is saves the HTML fragment to be sent back to the browser. The first parameter names the ID of the element whose HTML is to be replaced, and the second parameter contains the HTML.

INPUTS


  • string $parm1 - the html id
  • string $parm2 - the html value

SOURCE

function x6HTML($parm1,$parm2) { return x4html($parm1,$parm2); }
function x4HTML($parm1,$parm2) {
    if(!isset($GLOBALS['AG']['x4']['html'][$parm1])) {
        $GLOBALS['AG']['x4']['html'][$parm1] = '';
    }
    $GLOBALS['AG']['x4']['html'][$parm1] .= $parm2;
}
