NAME

String.htmlDisplay

FUNCTION

The Javascript function htmlDisplay converts the HTML characters ampersand '&', less than '<' and greater-than '>' to their HTML entities equivalents for safe display.

The reverse function is htmlEdit.

SOURCE

String.prototype.htmlDisplay = function() {
    if(this==null)  return '';
    return this.replace(/&/g,'&').replace(//g,'>');
}
