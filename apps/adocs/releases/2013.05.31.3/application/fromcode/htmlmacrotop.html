NAME

htmlMacroTop

FUNCTION

The PHP function htmlMacroTop is a shortcut for creating a top-level div that has class "fadein" (so that it fades in after the page is initialzied) and contains an H1 taken from the page or menu entry's title from the data dictionary.

INPUTS

string $page - The page, which is a table name or menu entry

RETURNS

androHtml DIV object with a title built in and having class "fadeIn".
