NAME

androHtmlxrefs

FUNCTION

The PHP class androHtmlxRefs generates a tabbed list of child tables to the named parent. Only child tables with the "x6xref" property set are included. The only supported value for "x6xref" at this time is "checkboxes".

The object is a subclass of androHtml, and supports all of its methods such as addChild, addClass, etc.

IMPORTANT! If there are no qualifying child tables, the object is still created and returned, but it will have "display: none" and will effectively not exist for the user.

INPUTS


  • string table_id - the parent table
  • number height - the total height available for the display

RETURNS

object - androHTMLxrefs object.
