NAME

x6Debug

FUNCTION

The PHP function x6Debug saves a debug datum to be returned to the browser. The browser takes no action with these items, they are meant to be inspected using Firebug or a similar tool.

INPUTS

string $parm1 - the debug info to save

SOURCE

function x6Debug($parm1) { return x4Debug($parm1); }
function x4Debug($parm1) {
    $GLOBALS['AG']['x4']['debug'][] = $parm1;
}
