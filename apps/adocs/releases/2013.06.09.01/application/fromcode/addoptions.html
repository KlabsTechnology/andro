NAME

addOptions

FUNCTION

The PHP Method addOptions takes an array of rows and creates on HTML OPTION object for each row. These are added to the parent object, which is assumed to be an HTML SELECT object.

INPUTS

array - an array of rows string - name of column to use as value string - name of column to use as display

SOURCE

function addOptions($rows,$value,$desc) {
    foreach($rows as $row) {
        $opt = $this->h('option',$row[$desc]);
        $opt->hp['value'] = $row[$value];
    }
}
