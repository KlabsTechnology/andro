NAME

androHtml.link

FUNCTION

The PHP method link adds a hyperlink to his androHtml object.

INPUTS

string $href - Hypertext reference string $innerHTML - inner html for the 'a' tag

SOURCE

function link($href,$innerHTML) {
    $a = $this->h('a',$innerHTML);
    $a->hp['href'] = $href;
    return $a;
}
