NAME

x6.json.newWindow

FUNCTION

The Javascript method x6.json.newWindow takes a JSON request and executes it as a page request, popping the result up in a new tab or window.

When the user exits the resulting tab or window, it will close.

EXAMPLE

The following example loads a new page

      <script>
      x6.json.init('x4Page','calendar');
      x6.json.newWindow();
      </script>

SOURCE

this.newWindow = function() {
    var entireGet = 'index.php?'+this.makeString()+'&x4Return=exit';
    $a.openWindow(entireGet);
}
