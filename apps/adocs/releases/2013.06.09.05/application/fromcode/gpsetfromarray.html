NAME

gpSetFromArray

FUNCTION

The PHP function gpSetFromArray adds all variables in the provided array into the Get/Post variables. In the array, the keys are the names of the variables, while the values are the values for the variables. If a prefix is provided, the prefix will be appended to the beginning of every variable name that is added.

INPUTS

string $prefix - Prefix to add at the beginning of each variable name array $array - Variables and their values to add into the Get/Post 

              parameters.

SOURCE

function gpSetFromArray($prefix,$array) {
   foreach($array as $key=>$value) {
      gpSet($prefix.$key,$value);
   }
}
