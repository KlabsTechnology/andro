NAME

x6dialogs.alert

FUNCTION

The Javascript method x6dialogs.alert replaces the native Javascript alert() function with one that is stylistically consistent with the rest of the application.

Unlike Javascript's native alert() function, execution continues after you call this function. The user must clear it by clicking the OK button, hitting Enter, or hitting Esc.

When this alert is active, all keyboard events are suppressed except Enter and Esc.

EXAMPLE

Here is a usage example:

       x6dialogs.alert("New data has been saved");
       // maybe do some other stuff while
       // waiting for the user
       u.events.notify('myEventName',objParms);
