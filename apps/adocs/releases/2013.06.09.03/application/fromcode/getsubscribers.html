NAME

x6events.getSubscribers

FUNCTION

The Javascript method x6events.getSubscribers returns an array of subscribers to a particular event.

NOTES

Use this method to discover which objects are subscribed to a particular event.

RETURNS

An array of zero or more object id's.

SOURCE

getSubscribers: function(eventName) {
    return x6.p(this.subscribers,eventName,[]);
},
