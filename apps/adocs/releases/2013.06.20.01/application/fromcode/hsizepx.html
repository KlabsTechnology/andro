NAME

hSizepx

FUNCTION

The PHP function hSizepx computes a width by examining the size cookie. It assumes that the baseline is 1024, and returns a string of the form "999px" that is scaled up or down based on what skin the user is actually using.

This function returns a string with 'px' appended.

INPUTS

number - Size in 1024x768 mode

SOURCE

function hSizepx($x1024) {
    $app  = $GLOBALS['AG']['application'];
    $size = a($_REQUEST,$app."_size",'1024');
    $final = intval(($x1024 * $size)/1024);
    return $final.'px';
}
