FUNCTION

The method nbsp adds the provided number of non-breaking spaces to this androHtml object as children elements.

INPUTS

number $count - Number of non-breaking spaces to add.

SOURCE

function nbsp($count=1) {
    for($x=1;$x<=$count;$x++) {
        $this->children[] = ' ';
    }
}
