JQuery Integration

Andromeda is a complete application framework. However, there is no reason for any framework author(s) to reinvent every software development tasks. Those tasks relating to the database are where Andromeda focuses its innovations, for the rest of the picture we seek to use the best of what others have done. This lets the Andromeda contributors focus on our specialties, and still gain the advantage of cutting edge packages made by others.

On the browser side we feel that JQuery is the best combination of power and elegance. JQuery is extremely easy to get started with, is supported by loads and loads of plugins, and is well documented.

JQuery homepage.

Default Features

Andromeda by default includes these libraries:

  • jquery-1.2.3.js
  • jquery.dimensions.js
  • jquery.tooltip.js (and jquery.tooltip.css)

Andromeda activates the tooltip feature for all A elements and INPUT elements. Please see the tooltip documentation (link above) for more options. This feature can be modified by the [[TOOLTIPS]] system variable.

Adding Your Own Libraries or Calls

If you want to make use of libraries that Andromeda has not included, you can do so quite easily. Use the [[cssInclude]] and [[jqPlugin]] functions to include your libraries. These can occur anywhere in your code. Andromeda will load the CSS in the document head and the JS at the bottom of the document. Here is a simple example: 

<?php
jqPlugin('appclib/jquery.example.js');
cssInclude('appclib/jquery.example.css');
?>

It is pretty common to need to run code when the document finishes loading. JQuery uses a call to "$(document).ready()" to do this. Andromeda provides a wrapper to this definition so that your application functions will not collide with framework functions. Any code that you need to run when the document is ready should be submitted with the [[elementAdd]] function, as in: 

<?php
elementAdd("jqueryDocumentReady","$('a').tooltip({showUrl: false})");
?>

Minification and YSlow!

Andromeda provides automatic combination and minification of Javascript and CSS files. More information is found on our pages on [[Andromeda HTML Templates]] and the [[JS_CSS_DEBUG]] system variable.
