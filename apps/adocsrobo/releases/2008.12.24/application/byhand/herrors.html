hErrors

string CSS_Class
string HTML_Fragment


Returns an HTML DIV element containing all reported errors. Each error is in an HTML P element.

You can specify a CSS Class to assign to the DIV element. If nothing is specified, the CSS Class "errorbox" is used.

Your CSS that defines the errors might look like this:

div.errorbox { ...properties for the errorbox.. } div.errorbox p { ...properties for each error entry }
