gp

Returns [[GET-POST Variable]], or a default value if the variable does not exist. 

string gp(  string varname
           [,string default_value]
         )

If varname was received as a GET or POST variable, the value is returned. If varname was not passed, the function returns either an empty string or the second parameter.

The following example returns the value of gp_page, or the value 'home' if gp_page was not passed. 

$gp_page = gp('gp_page','home');
