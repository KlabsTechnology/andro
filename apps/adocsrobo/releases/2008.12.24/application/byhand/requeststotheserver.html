Requests to The Server

The x4 javascript code running in the browser will send only three kinds of requests to the web server. One of these is a page request, and the other is a database command, and the third is a custom request. There are four database commands, so the complete picture is five commands:

  • A Page request

  • A Custom Request (not yet implemented).

  • An Insert

  • An Update

  • A Delete

  • A Select

The Return Structure

All requests return a JSON-encoded string. The string is eval'd and assigned to the global variable x4httpData. This variable is an associative array that contains the complete and entire response from the server. The x4httpData object is completely replaced on each call. The structure of the object never changes, though not all elements are present on each return.
