hWidget

string type_id
string name
string value (optional)


This function is new as of 3/9/07, and not yet fully populated. At the time of its creation, all widget generation is in [[ahInputsComprehensive]], with no ability to generate individual widgets as needed. This will be added to as needed to supply the various types.

If the type_id is cbool, then the HTML "value" property is always Y, and the third parameter is taken to be the caption.
