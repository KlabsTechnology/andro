Setting up apache and php via XAMPP

Be sure you completed all the steps in [[Running On Windows]] before proceding to these steps

Installation

  • Install the "XAMPP" package with default options. XAMPP for windows may be downloaded from Apache Friends.
  • Stop and disable the World Wide Web publishing service in the Services msc plugin (Goto Start | Run and type "services.msc"). You probably want to stop and disable all the other IIS services including the IIS Admin service unles you will be using them. It might be a good idea to uninstall these services.
  • Turn on Apache using the XAMPP control panel.
